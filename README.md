# Movie Recommendation System

This project recommends movies based on genre similarity using Machine Learning techniques.

## Dataset
A dataset of 30 movies with their genres was used.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Methodology
- Text vectorization using CountVectorizer
- Similarity calculation using cosine similarity
- Content-based recommendation system

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Enter a movie name to get recommendations
